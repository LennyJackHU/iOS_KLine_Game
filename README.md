A KLine game on ios. built with vibecoding. uses lightweightchart package.
this proj mainly uses an ipad and an esp32 to interact. using BLE, the ipad let esp32 control a coin giver, a coin receiver and a printer to simulate crypto trading by putting and receiving game coins.
data is random backtesting data abstract from yahoo finance,
